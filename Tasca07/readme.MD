# Tasca07: Eina d’assistència remota per a Helpdesk (EverPia)

OBJECTIU
--------
Analitzar eines d’assistència remota sota demanda i proposar-ne una
com a estàndard oficial d’EverPia per al suport immediat a usuaris finals.

CONTEXTE
--------
- El Helpdesk és una part clau de les hores facturables.
- Els usuaris NO poden:
  - Configurar VPNs
  - Obrir ports (RDP, 3389, etc.)
- Necessitem una eina que funcioni en segons, fins i tot en xarxes restrictives.

REQUISITS DE L’EINA
-------------------
- Connexió sota demanda (sense instal·lacions complexes)
- Fàcil per a l’usuari final
- Segura (xifrat, control d’accés)
- Ràpida i fiable
- Compatible amb entorns corporatius restrictius

TREBALL A REALITZAR (EN PARELLES)
--------------------------------
1. Analitzar el mercat d’eines de suport remot
   - Exemples: AnyDesk, TeamViewer, Quick Assist, RustDesk, etc.

2. Comparar-les segons:
   - Facilitat d’ús
   - Seguretat
   - Cost / llicències
   - Compatibilitat (Windows, macOS, Linux)
   - Experiència de l’usuari final

3. Proposar UNA eina com a estàndard oficial d’EverPia

4. Crear documentació:
   - Guia per a tècnics (com connectar-se, bones pràctiques)
   - Guia per a clients (passos simples per rebre suport)

RESULTAT FINAL
--------------
✔ Proposta justificada de l’eina escollida  
✔ Documentació clara per a tècnics i clients  
✔ Preparació per a suport Helpdesk real i eficient
